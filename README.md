# 🎓 Pós-Graduação .NET - NextWave Education

Repositório contendo o conteúdo das aulas e projetos desenvolvidos durante a Pós-Graduação em .NET pela **[NextWave Education](https://nextwave.education/)**.

## 📚 Estrutura do Repositório

As aulas e seus respectivos conteúdos estão organizados em diferentes **branches**, cada uma representando um módulo ou tema específico do curso.

## 📝 Requisitos do Projeto

- .NET SDK 10.0+
- Git
- Visual Studio, VS Code ou Rider
- Postman (para testes de API)
